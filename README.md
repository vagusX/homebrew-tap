# homebrew-tap

Homebrew formulae for [vagusX](https://github.com/vagusX) projects.

## Install

```bash
brew install vagusX/tap/jq5
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [jq5](https://github.com/vagusX/jq5) | An extension of jq to work on JSON5 files, preserving comments |
